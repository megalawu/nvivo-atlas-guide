Panduan Step-by-Step NVivo & ATLAS.ti

Mengapa Menggunakan NVivo?

NVivo adalah perangkat lunak terkemuka untuk penelitian kualitatif dan metode campuran. Software ini dirancang untuk membantu Anda mengatur, menganalisis, dan menemukan wawasan mendalam dalam data yang tidak terstruktur atau kualitatif seperti wawancara, survei terbuka, artikel, konten media sosial, dan video.

Panduan interaktif ini akan membawa Anda melalui alur kerja (workflow) dasar menggunakan NVivo, mulai dari mengatur proyek hingga memvisualisasikan temuan Anda.

Organisasi Data

Pusatkan semua data penelitian Anda (dokumen, audio, video) dalam satu platform yang terstruktur.

Analisis Mendalam

Gunakan fitur coding yang kuat untuk mengidentifikasi tema, pola, dan hubungan dalam data yang kompleks.

Validitas Temuan

Lacak kembali temuan ke data asli dengan mudah, memperkuat transparansi dan rigor penelitian Anda.

Fase 1: Persiapan & Import Data

Tutorial NVivo untuk Pemula: Pengenalan dan Cara Import Data Transkrip

Fokus: memahami antarmuka dasar NVivo dan alur import data (Word/PDF/transkrip) ke dalam proyek.

1

Membuat Proyek Baru

Langkah pertama adalah membuat ruang kerja digital untuk penelitian Anda.

  1. Buka aplikasi NVivo di komputer Anda.
  2. Pada layar awal (Welcome Screen), klik tombol Blank Project.
  3. Jendela New Project akan muncul. Ketik judul proyek Anda pada kolom Title.
  4. (Opsional) Isi kolom Description.
  5. Klik OK atau Create (tergantung versi NVivo).
2

Import Sumber Data (Files)

Langkah memasukkan file wawancara atau dokumen ke dalam proyek:

  1. Di panel navigasi kiri, pastikan Anda berada di area Data > Files.
  2. Pergi ke menu pita (Ribbon) di bagian atas layar, klik tab Import.
  3. Klik ikon Files (ikon berbentuk kertas/dokumen).
  4. Jendela browser file komputer akan terbuka. Cari dan pilih file (Word, PDF, dll) yang ingin di-import.
  5. Klik Open, lalu klik Import pada kotak dialog yang muncul.
Tips: Klik kanan di area putih pada panel Files, pilih New Folder untuk membuat folder pengelompokan (misal: Wawancara, Literatur).
3

Import Audio/Video + (Opsional) Transcript

Jika data Anda berupa rekaman, NVivo bisa menyimpan media dan (jika tersedia) transkripnya untuk memudahkan coding.

  1. Masuk ke Data > Files lalu klik tab Import.
  2. Pilih Audio atau Video, lalu pilih file rekaman Anda.
  3. Jika Anda punya transkrip (Word), import transkripnya sebagai file terpisah.
  4. Gunakan fitur Transcription/Sync (nama menu bisa berbeda antar versi) untuk menautkan transkrip dengan media.
Catatan: Nama menu bisa berbeda antar NVivo Windows/Mac. Prinsipnya sama: simpan media, lalu tautkan/selaraskan transkrip agar kutipan bisa diputar ulang saat membaca.
4

Membuat Cases & Memberi Atribut (Untuk Perbandingan)

Kasus (Cases) memodelkan unit analisis (misal: responden). Atribut (umur, gender, lokasi) membuat Anda bisa membandingkan tema antar kelompok.

  1. Siapkan daftar unit analisis (misal: Responden 1, Responden 2).
  2. Di panel navigasi, cari area Cases (atau Codes > Cases tergantung versi).
  3. Buat Case untuk tiap unit (klik Create > Case atau klik kanan > New).
  4. Buat Case Classification (misal: “Responden”) dan tambahkan field atribut (misal: Umur, Gender, Profesi).
  5. Isi atribut pada setiap Case, lalu tautkan Case ke sumber data (transkrip) agar analisis matriks berjalan.

Fase 2: Proses Coding (Pengkodean)

Coding adalah jantung dari analisis kualitatif. Ini adalah proses menandai segmen data dengan 'Node' (atau 'Code') yang mewakili tema, konsep, atau ide tertentu.

Cara Mudah Melakukan Coding Analisis Data Kualitatif Menggunakan NVivo

Fokus: memblok teks pada transkrip, membuat Node (kode), dan menyusun hirarki tema.

Membuat Codes (Pengkodean)

Cara menandai teks secara spesifik:

  1. Klik dua kali pada file dokumen di area Files untuk membukanya di layar kerja (Detail View).
  2. Gunakan mouse untuk memblok (highlight) kalimat yang penting.
  3. Klik kanan pada teks yang diblok tersebut.
  4. Pilih Code... atau Code Selection.
  5. Pilih opsi New Node (atau Top Level Code).
  6. Ketik nama tema di kolom Name (misal: "Kecepatan Layanan"), lalu klik OK.

Membangun Hirarki (Parent/Child)

Cara mengatur Node ke dalam folder hirarki:

  1. Di panel navigasi kiri, klik folder Codes > Nodes (atau sekadar Codes di versi baru).
  2. Anda akan melihat daftar semua Node yang sudah dibuat.
  3. Klik dan tahan klik (hold) pada Node anak (misal: "Kesopanan").
  4. Geser (drag) dan lepaskan (drop) tepat di atas teks Node induk (misal: "Kualitas Layanan").
  5. Sekarang "Kesopanan" akan menjorok ke dalam sebagai sub-tema di bawah "Kualitas Layanan".

Memos & Annotations (Catatan Analitik)

Cara Membuat Memo:

  1. Di menu Ribbon atas, klik tab Create.
  2. Klik ikon Memo.
  3. Ketik nama memo (misal: "Jurnal Riset Hari 1"), klik OK.
  4. Sebuah dokumen kosong terbuka, silakan ketik refleksi atau pemikiran teoritis Anda di situ.

Cara Membuat Annotation:

  1. Buka transkrip wawancara, blok kata/kalimat yang membingungkan atau menarik.
  2. Klik kanan pada teks yang diblok > pilih New Annotation.
  3. Ketik catatan singkat Anda di panel biru yang muncul di bagian bawah layar. Teks asli akan ditandai dengan warna biru.

Fitur Canggih: Auto Coding

Untuk dataset besar (seperti hasil survei ekstensif atau sentimen media sosial), gunakan fitur otomatisasi untuk menghemat waktu:

  • Auto Code by Theme/Sentiment: NVivo dapat mendeteksi tema umum secara otomatis (khusus versi Plus/Windows).
  • Auto Code by Structure: Sangat berguna untuk transkrip wawancara yang menggunakan Heading 1 untuk pertanyaan, Heading 2 untuk jawaban, sehingga memisahkan respons berdasarkan pertanyaan dengan cepat.

"Coding manual memberikan kedalaman konteks, Auto coding memberikan kecepatan eksplorasi awal."

Coding di Audio/Video (Jika Data Berupa Rekaman)

  1. Buka file audio/video di Files.
  2. Putar media, lalu pilih rentang waktu atau teks transkrip yang relevan (jika transkrip sudah ditautkan).
  3. Klik kanan > Code ke Node yang sesuai atau buat Node baru.
  4. Gunakan Annotations untuk mencatat konteks (intonasi, jeda, emosi) yang tidak tertangkap oleh teks.

Fase 3: Eksplorasi & Analisis Lanjutan

Setelah data dikodekan, saatnya bertanya pada data Anda menggunakan fitur Queries.

Ganti ID_VIDEO_DISINI dengan ID video YouTube yang Anda pilih.

Langkah Text Search Query

  1. Pada menu Ribbon atas, klik tab Explore.
  2. Klik ikon Text Search (ikon kaca pembesar).
  3. Pada jendela tab yang muncul, ketik kata yang ingin dicari di kolom Search for:.
  4. (Opsional) Geser slider Finding matches ke kanan (ke arah "Stemmed words" atau "Synonyms") jika ingin pencarian yang lebih luas.
  5. Klik tombol Run Query di bagian atas.
  6. Hasil teks yang mengandung kata tersebut akan muncul. Untuk menjadikannya Node, klik tombol Save Results di ribbon atas.

Langkah Word Frequency Query

  1. Pada menu Ribbon atas, klik tab Explore.
  2. Klik ikon Word Frequency.
  3. Biarkan default pengaturan (untuk mencari di semua file), lalu klik tombol Run Query.
  4. Daftar kata yang sering muncul akan dimuat. Untuk melihat visualnya, klik tab Word Cloud di sebelah kanan atas area hasil pencarian.
  5. Tips: Jika muncul kata seperti "yang" atau "dan", klik kanan pada kata tersebut di tabel hasil > pilih Add to Stop Words List > klik OK, lalu klik Run Query lagi.

Langkah Matrix Coding Query

  1. Pada menu Ribbon atas, klik tab Explore.
  2. Klik ikon Matrix Coding.
  3. Pada pengaturan Rows (Baris), klik tombol + lalu pilih Select Items.... Centang Nodes (Tema) yang ingin dibandingkan, klik OK.
  4. Pada pengaturan Columns (Kolom), klik tombol + lalu pilih Select Items.... Centang Atribut Demografi atau Cases, klik OK.
  5. Klik Run Query.
  6. Tabel matriks dengan angka silang akan muncul. Anda bisa klik dua kali (double-click) pada salah satu angka di tabel untuk langsung melihat kutipan aslinya.

Penting: Gunakan "Cases" dan "Classifications"

Agar analisis Anda lebih bermakna, ubah narasumber/unit analisis Anda menjadi Cases. Kemudian, berikan atribut pada Cases tersebut menggunakan Case Classifications (misal: Case = Responden 1; Atribut = Umur 25, Profesi Guru).

Ini adalah syarat wajib agar Anda bisa melakukan Matrix Coding Query untuk membandingkan demografi!

Fase 4: Visualisasi & Pelaporan

Memvisualisasikan data kualitatif membantu menemukan pola baru dan mempresentasikan temuan Anda secara menarik di laporan akhir.

Video Tutorial: Membuat Visualisasi Data (Word Cloud, Chart, & Mapping) di NVivo

Ganti ID_VIDEO_DISINI dengan ID video YouTube yang Anda pilih.

Inovasi Cepat Kualitas Solusi

Word Cloud

Representasi visual dari frekuensi kata. Kata yang lebih besar berarti lebih sering muncul.

Charts (Grafik)

Buat grafik batang atau pie untuk menunjukkan jumlah referensi coding pada Node tertentu, atau untuk melihat demografi (Cases).

Mind Maps & Concept Maps

Alat brainstorming di dalam NVivo. Mind Map untuk ide awal, Concept Map untuk menghubungkan teori dan temuan antar Nodes.

Hierarchy Charts (Tree Map)

Kotak-kotak bersarang yang ukurannya menunjukkan jumlah coding. Sangat bagus untuk membandingkan dominansi antar tema (Nodes).

Semua visualisasi dapat di-eksport (sebagai gambar atau tabel) untuk dimasukkan ke laporan.

Checklist Ekspor yang Paling Sering Dipakai

  • Ekspor visual (word cloud, chart, map): cari tombol Export di ribbon pada tampilan visual.
  • Ekspor daftar Nodes/Codes: buka daftar Nodes > gunakan Export List (jika tersedia) atau salin tabel.
  • Ekspor hasil query (Text Search/Matrix): dari hasil query > Save/Export untuk disimpan sebagai item proyek atau file.
  • Ekspor kutipan (coded references): buka Node > tampilkan referensi > simpan/ekspor sebagai dokumen untuk lampiran.

Fase 1: Persiapan Proyek & Dokumen (ATLAS.ti)

Langkah awal untuk memulai riset Anda di ATLAS.ti. Dalam ATLAS.ti, sumber data Anda disebut sebagai “Documents”.

Belajar ATLAS.ti Dasar: Cara Membuat Proyek Baru dan Menambahkan Dokumen

Fokus: membuat project baru dan mengimpor dokumen ke Document Manager agar siap dianalisis.

1

Membuat Proyek Baru

  1. Buka aplikasi ATLAS.ti.
  2. Pada layar Welcome, klik Create New Project.
  3. Masukkan Project Name dan (opsional) Comment.
  4. Klik Create.
2

Menambahkan Dokumen (Import)

  1. Buka tab Home.
  2. Klik Add Documents > pilih Add Files....
  3. Pilih file (Word, PDF, audio, video) > klik Open.
  4. Dokumen muncul di area Project Explorer di sebelah kiri.

Membuat Document Groups (Pengelompokan Data)

Kelompokkan dokumen (misal: Pria vs Wanita, Wawancara vs Literatur) agar analisis perbandingan lebih mudah.

  1. Klik Documents untuk membuka Document Manager.
  2. Klik kanan di area group > New Group > beri nama.
  3. Pilih dokumen di daftar > drag and drop ke group yang sesuai.

Menambahkan Document Variables (Atribut Dokumen)

Jika Anda ingin membandingkan tema antar kelompok, buat variabel seperti Gender, Usia, atau Lokasi pada dokumen/responden.

  1. Buka Document Manager, lalu cari menu Document Variables (nama bisa berbeda antar versi).
  2. Buat variabel (misal: Gender) dan pilih tipe nilainya (teks/angka).
  3. Isi nilai variabel untuk tiap dokumen agar bisa dipakai di tabel analisis.

Fase 2: Proses Coding & Memos (ATLAS.ti)

Coding memecah data dengan memberi label (Codes) pada kutipan (Quotations) yang relevan.

Tutorial Coding ATLAS.ti: Open Coding, In-Vivo, dan Pembuatan Memos

Fokus: open coding, in-vivo, serta menulis dan menautkan memos selama proses analisis.

■ Open Coding (Membuat Code Baru)

  1. Buka dokumen dari Project Explorer.
  2. Blok teks penting (ini menjadi Quotation).
  3. Klik kanan > Open Coding.
  4. Ketik nama code > klik Create.

■ List Coding & In-Vivo

List Coding (pakai code yang sudah ada):

  1. Blok teks > klik kanan > List Coding.
  2. Pilih code dari daftar > OK.

Code In-Vivo (kata asli jadi nama code):

  1. Blok kata/frasa penting.
  2. Klik kanan > Code in Vivo.

Menulis Memos & Menautkannya

Gunakan memos untuk mencatat ide teoretis dan keputusan analitik agar jejak analisis jelas.

  1. Membuat Memo Baru:
  2. Pada tab Home, buka menu Memos.
  3. Pilih Create Free Memo > beri nama > tulis catatan Anda.
  1. Menautkan Memo (Linking):
  2. Klik sebuah quotation atau code di margin.
  3. Dari panel Memos, drag and drop memo ke quotation/code.

Mengelola Daftar Code (Rename, Merge, Group)

  1. Buka Code Manager untuk melihat semua code.
  2. Rapikan nama: gunakan rename agar konsisten (misal: kata benda, bentuk tunggal).
  3. Jika ada duplikasi, gunakan merge (gabung) agar tidak terpecah.
  4. Buat Code Groups untuk menyusun tema (misal: “Hambatan”, “Dukungan”, “Strategi”).

Fase 3: Analisis & Queries (ATLAS.ti)

Cari pola, frekuensi kata, dan analisis persinggungan antar tema (co-occurrence).

Video Tutorial: Code Co-occurrence dan Penggunaan Query Tool di ATLAS.ti

Ganti ID_VIDEO_DISINI dengan ID video YouTube yang Anda pilih.

1. Word Cloud / Word Frequencies

Melihat kata yang sering muncul untuk eksplorasi awal.

  1. Buka tab Analyze.
  2. Klik Word Frequencies.
  3. Pilih dokumen/group yang dianalisis, lalu tampilkan word cloud.
  4. Tambahkan kata umum ke stop list agar hasil lebih bersih.

2. Code Co-occurrence Explorer

Menganalisis keterkaitan tema: di mana dua code muncul bersama.

  1. Buka tab Analyze.
  2. Pilih Code Co-occurrence Explorer.
  3. Pilih code untuk baris dan kolom, lalu baca sel matriksnya.
  4. Klik sel berangka untuk membuka kutipan aslinya.

3. Query Tool (Pencarian Pola/Aturan)

Gunakan query untuk mencari segmen yang memenuhi kriteria (misal: Code A AND Code B, atau Code A dalam kelompok dokumen tertentu).

  1. Buka tab Analyze > pilih Query Tool.
  2. Susun aturan (AND/OR/NOT), pilih cakupan dokumen/group.
  3. Jalankan query, lalu simpan hasil sebagai output atau item analisis.

Fase 4: Networks (Peta Konsep) (ATLAS.ti)

ATLAS.ti unggul dalam networks untuk memvisualisasikan hubungan antar codes, memos, dan dokumen.

Video Tutorial: Cara Membuat Peta Konsep (Network Canvas) di ATLAS.ti

Ganti ID_VIDEO_DISINI dengan ID video YouTube yang Anda pilih.

Langkah Membuat Network

Membangun model teoretis visual dari temuan.

Tahap 1: Membuka Canvas

  1. Buka tab Home.
  2. Buka menu Networks > pilih New Network.
  3. Beri nama network, lalu kanvas kosong terbuka.

Tahap 2: Menarik Node & Membuat Relasi

  1. Buka panel codes/memos/dokumen yang ingin dipetakan.
  2. Drag and drop item ke kanvas.
  3. Tarik konektor dari satu node ke node lain, lalu pilih jenis relasinya.
  4. Tambahkan label relasi agar argumentasi teoretis lebih jelas.
Ekspor: Simpan network sebagai gambar untuk laporan dengan menu Export > Save as Image (nama menu bisa berbeda antar versi).
